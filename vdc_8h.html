void pce_vdc_dma_start ( uint8_t  mode,
uint16_t  source,
uint16_t  dest,
uint16_t  length 
)

Start a DMA operation on the VDC.

Parameters
modeThe DMA mode.
  • VDC_DMA_IRQ_SATB_DONE - IRQ on VRAM->SATB transfer completion
  • VDC_DMA_IRQ_DONE - IRQ on VRAM->VRAM transfer completion
  • VDC_DMA_SRC_DEC - decrement source pointer (increment by default)
  • VDC_DMA_DEST_DEC - decrement destination pointer (increment by default)
  • VDC_DMA_REPEAT_SATB - automatically schedule VRAM->SATB transfer
sourceSource address in VRAM.
destDestination address in VRAM.
lengthLength, in bytes.
